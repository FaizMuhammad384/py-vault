# AppVault - Python File Manager

AppVault is a simple and efficient command-line File Manager application built with Python.
This project allows users to perform basic file handling operations such as creating, reading, writing, and deleting files directly from the terminal.

## Features

* Create new files
* Read file content
* Write data into files
* Delete existing files
* Simple command-line interface
* Exception handling for errors
